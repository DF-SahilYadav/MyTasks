URL parameters

All optional, all combinable.

?quickadd=1 — open quick-add modal
&text=your+task — pre-fill task name (URL-encoded)
&notes=details — pre-fill notes
&q=q1 (or q2/q3) — select quadrant
&d=1 — mark as deliverable
&r=1 — mark as daily recurring
&silent=1 — add directly without showing modal
?focus=q1 (or q2/q3) — open matrix with that quadrant's inline add focused

Hyperkey — open matrix with inline add ready

A hyperkey is Caps Lock remapped to ⌘⌃⌥⇧ via Karabiner-Elements or BetterTouchTool. Pair it with a Raycast Quicklink to open the matrix with a quadrant pre-focused — no modal, just start typing.
